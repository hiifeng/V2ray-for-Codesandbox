# 简介：
本项目用于在 CodeSandBox.io 免费服务上部署 V2ray ，采用的方案为 Nginx + WebSocket + VMess/VLess + TLS。晚高峰时速度不是很理想，建议配合 Cloudflare CDN 使用。
